Добро пожаловать в MathParser!

Этот сервис предназначен для парсинга и вычисления математических формул.

Поддерживаемые операторы: +, -, /,*.
Поддерживаемые функции:

  • ^ - возведение в степень;
  • e - число e;
  • pi - число "пи";
  • exp(n) - e^n
  • cos(x)
  • sin(x)
  • tg(x)
  • ctg(x)
  • log(a, b), где a - основание логарифма
Примеры выражений:
  • (1+2*cos(pi/2))^2
  • (x-y)*(cos(0)) (перед нажатием кнопки "Посчитать" добавьте параметры x и у)
  • 2 + 0.5 + 2.5*cos(pi) - log(2, 8) + sin(x) + tg(x)^2 (перед нажатием кнопки "Посчитать" добавьте параметр x)
